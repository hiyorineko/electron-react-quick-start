## What is this?
Electron + React + TypeScript Environment on Docker

## Quick start
### setup
```bash
$ git clone https://github.com/hiyorineko/electron-react-quick-start
$ cd electron-react-quick-start
$ make init
```

### container up
```bash
$ make up
```

## application-build
```bash
## build & watch
$ make app-build

## prod
$ make app-build-prod
```

## application-packaging
```bash
## Windows
$ make packaging-win

## macOS
$ make packaging-mac
```
